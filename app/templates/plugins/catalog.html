Write a plugin

A module: drop app/modules/yourmodule.py with a @register class extending Module. A parser: drop app/plugins/your_plugin.py with a @register_parser class extending ParserPlugin (a detect(), parse() and a render partial). No other file changes — it registers itself and shows up throughout the UI.
